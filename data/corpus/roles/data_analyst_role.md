# Role Guide: Data Analyst (Test-only)

## Core Responsibilities
- Create dashboards and weekly metrics reports
- Support ad-hoc analysis for product and ops teams
- Define and maintain metric definitions
- Ensure data quality checks on key pipelines (coordination with engineering)

## First 30 Days Expectations
- Learn key business metrics and where they are defined
- Deliver one metrics report and one dashboard update
- Document at least 5 metric definitions in the wiki

## Common Tools
- SQL + notebook environment
- BI dashboard tool
- Version control for analysis code (as applicable)

## Who to Ask
- Data lead for metric definitions
- Product manager for prioritization
- Engineering for pipeline access and data sources
