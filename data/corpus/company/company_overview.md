# Example Organization Overview (Test-only)

## Mission
Enable customers to make confident decisions using reliable products and responsive support.

## Values
- Customer-first
- Ownership
- Clarity and transparency
- Continuous learning
- Data-informed decisions

## Work Model and Hours
- Standard work hours: 10:00–19:00 local time (flexible with manager approval).
- Remote work: allowed up to 2 days/week after first 30 days (role-dependent).
- Core collaboration window: 12:00–16:00.

## Communication Norms
- Use the internal chat tool for quick questions.
- Use email for formal approvals and external communication.
- Document decisions in the project wiki within 24 hours.

## Org Structure (High-level)
- People Operations (HR)
- Engineering
- Product
- Data & Analytics
- Customer Support
- Finance & Procurement
- IT & Security

## Common Tools (References)
- Project wiki (knowledge base)
- Ticketing system (HR/IT requests)
- Expense portal
- Identity provider (SSO)
